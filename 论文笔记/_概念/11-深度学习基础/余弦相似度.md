# 余弦相似度

## 定义

两个向量夹角的余弦值，衡量方向上的相似性，与向量长度无关。值域 $[-1, 1]$，值越大表示方向越相似。

## 数学形式

$$
\cos(\mathbf{u}, \mathbf{v}) = \frac{\mathbf{u} \cdot \mathbf{v}}{\|\mathbf{u}\| \|\mathbf{v}\|}
$$

余弦距离（dissimilarity）常定义为 $d = 1 - \cos(\mathbf{u}, \mathbf{v})$，用于 token 去重/聚类。

## 核心要点

1. **尺度不变性**: 只关心方向，不受向量 L2 范数影响，适合归一化特征空间
2. **在[[视觉Token剪枝]]中的应用**: 以余弦相似度为准则剔除与已选 token 方向相近的冗余 token
3. **[[AgilePruner]] 中的用法**: 若候选 token 与当前最高注意力 token 的余弦距离 < $\tau_i$，则视为冗余并剪除
4. **与 L2 距离的对比**: 余弦距离不受 token 嵌入模长影响，更适合语义相似度判断

## 代表工作

- [[AgilePruner]]: 用自适应余弦相似度阈值 $\tau_i$ 筛选多样化视觉 token（ICLR 2026）
